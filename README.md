# SpringBoot-Examples

This repository contains multiple independent Spring Boot example projects.

## Projects

- `auth-tutorial` - Spring Security + JWT authentication tutorial app.
- `demo` - Basic Spring Boot REST + JPA sample app.
- `mobile-app-ws` - User-focused REST API project with security and API docs.
- `mobile-app-ws-rest-assured-test` - Rest Assured test suite for `mobile-app-ws` APIs.
- `mvc-example` - Spring MVC (Thymeleaf) web application example.
- `stockmarket` - Stock-market-themed Spring Boot data/web sample.
- `teknofest-backend` - Backend app with Thymeleaf and JPA.
- `verification-service` - Verification service web app module.
- `watercomp` - Water comparison project with geospatial dependencies.
- `watercompV2` - Second version of the water comparison project.

## Notes

Each top-level project directory includes its own `README.md` with module-specific details and run commands.
